CrazyStat Config-Dokumentation

Sie können alle diese Einstellungen in usr/config.php verändern oder hinzufügen. Wenn dort eine Einstellung nicht gesetzt ist, wird der Default-Wert verwendet.
Einstellungen können auch vom eingeloggten User abhängig gemacht werden.
Beispiel: $user_config["admin"]["stat_long_bars"]=false;
Diese Einstellung setzt $config_stat_long_bars für den Nutzer admin auf false. Nicht alle Einstellungen lassen sich sinnvoll nutzerabhängig machen (z.B. $config_stat_user).
Einstellungen, bei denen das vermerkt ist, können mittels $set_* überschrieben werden. So kann z.b. $config_counter_enabled mit folgendem Einbindungs-Code überschrieben werden:
<?php $set_counter_enabled=true; @include_once("stat/src/include.php"); ?>
