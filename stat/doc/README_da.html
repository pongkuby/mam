2. Egenskaber

Generelt

  • er i stand til at fungere helt uden JavaScript (hvilket mange scripts ikke er)
  • ingen reklamer, logoer eller lignende ting optræder på de sider, som tælles - kun en lille bemærkning i alt-attributten ved tilvalg af tæller
  • uafhængig af andre serviceudbydere, da der er talte om et PHP script, som kører på din server
  • konfigurering af IP-blokering
  • IPere kan gemmes anonymt, hvis ønsket, således at de besøgende ikke kan spores
  • statistik og tæller i et og samme script!
  • ingen MySQL nødvendig
  • adgang til serverens logfiler ikke nødvendig
  • dine sider vil fortsat være gyldig HTML eller XHTML
  • du kan vælge om dine egne besøg skal tælles med eller ej
  • log-filer kan komprimeres med gzip, hvis ønsket, så de bruger så lidt webplads som muligt

Statistik

  • Alle resultater vises i en overskuelig brugerflade, som ved hjælp af søjlediagrammer giver et hurtigt overblik over alle informationer (for browsere, skærmopløsninger og operativsystemer vises også cirkeldiagrammer).
  • Følgende data for hits og besøgende bliver analyseret både mht. tidsrum og totalt tidsforbrug:
    • hits i alt
    • hits denne måned
    • hits i forrige måned
    • brugere online
    • flest hits / dag
    • hits i snit / dag
    • hits / ugedag
    • hits / måned
    • hits / dag
    • hits / time
    • den browser som brugeren anvender
    • filer (sider) set af den besøgende
    • skærmopløsning
    • antal farver
    • operativsystem
    • henvisninger
    • de søgeord, som brugeren anvendte, før han/hun kom til hjemmesiden
  • gennemsnits -og sumværdier i alle moduler
  • vælg om du vil beskytte statistikken med forskellige adgangskoder for flere forskellige brugere - adgangskoder gemmes og sendes over nettet i krypteret form
  • individuelle "forudindstillinger" kan gemmes, så du har hurtig og let adgang til din foretrukne måde at få vist statistikken på
  • download og håndtér log-filer via web-brugerfladen (backup, slet)
  • se logfiler (som tabel)
  • søgning i log-filer (filter)
  • optimeret analyseringsevne takket være flere forskellige caching-teknikker
  • flere sprog (p.t. tysk, engelsk, hollandsk og dansk)

Tæller

  • kan deaktiveres
  • når tælleren er deaktiveret, ser den besøgende et billede, som du vælger (f.eks. blank.gif => brugeren ser ingenting)
  • der kan frit vælges et design for tælleren blandt hundredvis af gratis billeder
  • tæl hits eller besøgende (blokering af ip'ere)
  • vælg frit hvor mange tal, du vil have vist
  • mulighed for transparens, så din tæller kan integreres bedst muligt i dit design
  • en hvilken som helst værdi kan lægges til den rigtige tæller-værdi (f.eks. værdien af din gamle tæller)
  • du kan bestemme, hvilken tekst tælleren skal vise, når siden opdateres
  • i stedet for en billed-tæller kan du få vist en tekstbaseret tæller, som let lader sig integrere med en hvilken som helst tekst og hvis udseende kan defineres via CSS

3. Forudsætninger

Krav til serveren

  • PHP >= 5.1 (5.3.3, 5.3.8 er alle blevet testet), for tæller og cirkeldiagrammer kræves gdlib > 2.0.28.
    Af sikkerhedsmæssige hensyn anbefales generelt register_globals = Off (men dette er ikke et krav). Fungerer også ved safe_mode = On.
  • Ingen MySQL
  • Hjemmesider med en masse besøgende bør anvende en "stærk" server
  • Scriptet kører på stort set alle php-webservere, inkl. dem der er gratis

Krav til browseren

  • Browser (Internet Explorer 5.5 - 9, Opera 5 - 11.62, Firefox 1.0 - 11, Safari 3.2.1 - 5.1.5 (Win), Google Chrome 1.0 - 18 er blevet testet
  • Visning af billeder (må ikke være deaktiveret, tekstbrowsere bliver ikke talt med)
  • JavaScript (er kun) nødvendig for at opfange skærmopløsning og antallet af farver - plus nogle funktioner som er rare at have på siden med statistik (men selv statistik-siden fungerer uden JavaScript!)

4. Installation

Se FAQ hvis du har brug for hjælp i forbindelse med opdatering af CrazyStat 1.7x (inklusiv ekstra udgivelser) til denne version.
Opdatering fra CrazyStat 1.5x eller 1.6x er alene beskrevet i den tyske FAQ (fordi CrazyStat < 1.70 kun fandtes på tysk).

Hurtig installation (trin i parentes gælder kun tælleren!)

Denne guide er tiltænkt erfarne brugere.

  1. Pak filen ud
  2. (Vælg/download tæller-design (grafik findes i stat/usr/counter_styles/ - andre designs findes på http://en.christosoft.de/Crazystat/Counter_styles)
  3. (Konvertér tæller-designet til PNG (eller gif, hvis du har GDLib-understøttelse af gif) (se FAQ) og gem i "stat/usr/counter_styles")
  4. Åben stat/usr/config.php og tilpas indstillingerne (især $config_rel_path)
  5. Upload "stat"-mappen inklusiv alle filer og undermapper
  6. Anvend chmod 777 på stat/usr og alle mappens filer samt undermapper. Gør det samme med stat/src/log-tool
  7. Indsæt <?php @include_once("stat/src/include.php"); ?> mellem <body> og </body> på alle de sider, som skal tælles med, skift filendelse fra ".html" eller ".htm" til ".php" (hvis endelsen ikke i forvejen er ".php"). Se FAQ: "CrazyStat i .html-filer", "Tæl sider, som ligger i forskellige mapper"
  8. Åben statistikken (http://www.example.com/stat/). Hvis du har valgt at beskytte siden med adgangskode, bedes du ændre adgangskoden, når du logger ind. Har du ikke foretaget ændringer mht. brugernavn og adgangskode er brugernavnet "admin", adgangskoden er "pass".

Hvis du har ændret "stat" til noget andet, skal du tilpasse koden (trin 7) og $config_rel_path (trin 4)!

Detaljeret installationsproces med tæller

  1. Pak filen ud et eller andet sted på din egen harddisk (benyt f.eks. 7-Zip)
  2. Vælg et design til tælleren. Du finder forskellige designs i mappen "stat/usr/counter_styles", som er klar til brug. På http://en.christosoft.de/Crazystat/Counter_styles kan du finde endnu flere designs, links til sider med stadig flere designs såvel som tips til, hvordan du selv kan designe en tæller.
  3. Du skal sikre dig, at tælleren er gemt i PNG-format (eller GIF hvis din server understøtter det). I ofte stillede spørgsmål - eller FAQ - finder du instruktioner i, hvordan man konverterer sit tæller-design til PNG. Placér dit design i mappen "stat/usr/counter_styles".
  4. Åben "stat/usr/config.php" i et rent tekstprogram (såsom Notesblok eller PSPad) og tilpas indstillingerne efter dine behov (især $config_rel_path). Tællerens filnavn skal angives i $config_counter_file_name og for at aktivere tælleren skal der endvidere stå $config_counter_enabled=true;
    Mere information om alle konfigurationsindstillingerne
  5. Upload "stat"-mappen med alle filer og undermapper til din server (f.eks. via FTP med FileZilla)
  6. Anvend chmod 777 på mappen "stat/usr" inklusiv alle mappens filer og undermapper. Gentag med mappen "stat/src/log-tool". Se FAQ hvis du ikke ved, hvordan dette gøres.
  7. Se FAQ: "CrazyStat i .html-filer", "Tæl sider, som ligger i forskellige mapper"
