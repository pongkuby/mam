Documentation

Here you can find the documentation files of CrazyStat, an open source php script for visitor statistics with optional counter.
These files can be found in the doc-folder of the CrazyStat package and on documentation.crazystat.de. The files on the web might be newer than those that come with the current CrazyStat version.
